# 🛢️ Thar Oil - E-commerce Website

Thar Oil is a full-stack e-commerce website designed for selling premium **Kachhi Ghani Moongfali (Groundnut) Oil**.
This project focuses on purity, tradition, and a seamless online shopping experience.

---

## 🚀 Features

* 🛒 Add to Cart & Checkout System
* 👤 User Authentication (Login / Signup)
* 🔐 JWT-based Secure Login
* 🧑‍💼 Admin Panel
* 📦 Order Management
* 📰 Blog Section
* 📍 Dealer Inquiry System
* 📜 Policies Pages (Privacy, Terms, Delivery)
* 🌐 SEO Optimized (Sitemap included)

---

## 🛠️ Tech Stack

### Frontend:

* HTML5
* CSS3
* JavaScript

### Backend:

* Node.js
* Express.js

### Other Tools:

* JSON Database
* JWT (Authentication)
* bcryptjs (Password Security)
* Multer (File Uploads)

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/tharoil-website.git
```

### 2. Go to project folder

```bash
cd tharoil-website
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run server

```bash
npm start
```

### 5. Open in browser

```
http://localhost:3000
```
